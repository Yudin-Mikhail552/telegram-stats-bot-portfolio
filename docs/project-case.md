# Telegram Stats Bot — описание проекта

## Задача

В рабочих Telegram-группах ежедневно появляется большое количество заявок. Руководителям и сотрудникам нужно быстро понимать:

- сколько заявок пришло за день;
- какие категории лидируют;
- какие источники дают больше заявок;
- какие заявки оформлены некорректно;
- как меняется загрузка по дням и периодам.

Ручной подсчёт занимал время и давал риск ошибок.

## Решение

Был внедрён Telegram-бот, который автоматически собирает заявки, сохраняет их в базу, группирует по категориям и источникам, а затем формирует отчёты по командам и расписанию.

## Основные функции

- сбор заявок из рабочих Telegram-групп;
- сохранение данных в SQLite;
- обновление данных при редактировании сообщений;
- группировка по категориям;
- группировка по источникам;
- отчёты за день, неделю, месяц и выбранную дату;
- список проблемных заявок;
- ежедневная автоматическая отправка отчётов;
- проверка состояния через команду /ping.

## Результат

Бот сократил ручной подсчёт заявок, повысил прозрачность работы офиса и дал руководителям быстрый инструмент для контроля загрузки, источников и категорий.
